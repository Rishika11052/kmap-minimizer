##Installation and Setup

# Clone the repository
```
git clone 'https://github.com/Rishika11052/kmap-minimizer.git' 
```
```
cd kmap-minimizer
```

# Install dependencies
```
npm install
```

# Start development server
```
npm run dev
```

# Build for production
```
npm run build
```

# Preview the production build
```
npm run preview
```
